# Quickstart: Theme Management

**Branch**: `001-add-theme-management` | **Date**: 2025-09-09 | **Spec**: [./spec.md](./spec.md)

## 1. Prerequisites
- You must be logged in as a "Station Manager".

## 2. Steps to Test
1.  Navigate to the "Settings" page.
2.  Find the "Theme" section.
3.  Select a new primary color using the color picker.
4.  Switch between "Light" and "Dark" mode.
5.  Click "Save".

## 3. Expected Results
- The application's theme should update instantly to reflect the changes.
- The selected theme should persist across page reloads.
- Other users of the same station should see the new theme.
